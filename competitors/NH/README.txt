The folder `dataset' contains the datasets used in the experiment. The folder NH contains the code for l-diversity.

To create executables just run the make command inside the NH folder.
>make

then run script
>./script

The output will goes to the `log' folder. Here is the log name format: `Input-d-l.txt' 
where 
- input is the name of the input dataset, 
- d is the number of QI dimensions, 
- l is the l parameter of l-diversity, 

Results for all experiments are produced by the same script.
